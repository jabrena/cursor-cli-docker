# Develop a HelloWorld Java Class program

## Role

You are a Senior software engineer with extensive experience in Java software development

## Goal

Develop a classic Java class HelloWorld.java program
that print "Hello World" in the console only,
Follow the next steps to achieve the goal.

## Steps

### Step 1: Develop the HelloWorld Java Class program
        
#### Step Constraints

- Develop the class in the Maven module `sandbox`
- Develop the class in the package info.jab.examples
- Don't invest time in planning
- Don't create any test class
- Don't touch the build file (pom.xml)

## Output format

- Explain the process

## Acceptance Criteria

The goal will be achieved if the following criteria are met:

- The solution is compiled successfully with `./mvnw clean compile -pl sandbox`
- The solution only prints "Hello World" in the console
- The solution is committed and pushed to the branch to create the PR
- Only commit the file HelloWorld.java only and push the changes to the branch to create the PR
- The output is in the correct format
